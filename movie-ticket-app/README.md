# 🎬 Movie Ticket Booking App

A full-stack Movie Ticket Booking Application built using:

- Spring Boot
- Thymeleaf
- Spring Security
- JPA/Hibernate
- MySQL
- Bootstrap
- Git & GitHub

## Features

- Add Movies
- Edit Movies
- Delete Movies
- Search Movies
- Validation
- Movie Details Page
- Dynamic Poster Rendering

## Tech Stack

- Java 17
- Spring Boot
- MySQL
- Bootstrap 5
- Thymeleaf
